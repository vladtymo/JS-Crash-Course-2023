About Git Clone Command

NAME

git-reset - Reset current HEAD to the specified stat

DESCRIPTION

In the first three forms, copy entries from to the index. In the last form, set the current branch head (HEAD) to, optionally modifying index and working tree to match. The defaults to HEAD in all forms. These forms reset the index entries for all paths that match the to their state at. (It does not affect the working tree or the current branch.)

SYNOPSIS

git reset [-q] [<tree-ish>] [--] <pathspec>…​ git reset [-q] [--pathspec-from-file=<file> [--pathspec-file-nul]] [<tree-ish>] git reset (--patch | -p) [<tree-ish>] [--] [<pathspec>…​] git reset [--soft | --mixed [-N] | --hard | --merge | --keep] [-q] [<commit>]
