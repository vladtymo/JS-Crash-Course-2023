Git Tag Command

The git tag command is used to create, list, delete, and verify tags in a Git repository.

Creating Tags

To create a tag in Git, you can use the following command:

git tag <tagname>

This will create a new tag with the given name at the current commit.

Listing Tags

To list all tags in a Git repository, you can use the following command:

git tag

This will list all tags in alphabetical order.

Deleting Tags

To delete a tag in Git, you can use the following command:

git tag -d <tagname>

This will delete the tag with the given name.

Verifying Tags

To verify the validity of a tag in Git, you can use the following command:

git tag -v <tagname>

This will verify the tag's signature and display information about it, such as the tagger and commit message.
